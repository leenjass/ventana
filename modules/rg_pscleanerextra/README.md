Rolige PrestaShop Cleaner Extra
==================================================
## Description
Clean obsolete data, check and fix functional integrity constraints, rename DB prefix, chance tables to InnoDB engines, clear obsolete product images and remove default data.

## Requirements
- [PrestaShop 1.6.x - 1.7.x](https://www.prestashop.com/)
- [PHP 5.4+](http://www.php.net/)
